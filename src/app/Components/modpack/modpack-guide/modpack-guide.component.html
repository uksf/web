Modpack Setup Guide

  • Welcome to the UKSF Modpack Setup Guide
  • Please follow the steps carefully
  • If you need help, please contact an SR1

    •

    Arma3Sync

    Download Arma3Sync

  • Install wherever you prefer
  • Right click Arma3Sync.exe or the shortcut
  • Select 'Properties'
  • Check 'Run this program as an administrator'

    •

    Arma3Sync Setup

  • Set executable location to arma3_x64.exe
  • Set addon search directory to arma 3 location

    •

    Adding Repository

  • Click 'Repositories'
  • Click blue plus
  • Paste https://modpack.uk-sf.co.uk/.a3s/autoconfig into 'Public auto-config url'
  • Set name as 'UKSF'
  • Press 'Import' and 'OK'

    •

    Connect To Repository

  • Select 'UKSF' in the list
  • Press 'Connect' (Page with blue arrow)

    •

    Check Repository

  • Press 'Check for Addons' (Green Tick)
  • Press 'Settings'
  • Set 'Maximum number of connections' to 10

    •

    Download Repository

  • Look in the 'Repository content' panel
  • Any entry in red or with a yellow triangle indicates a missing or incomplete mod
  • Press 'Select All'
  • Press 'Start'
  • Allow time for the mods to download

    •

    Create Modset

  • Make sure mods are fully downloaded
  • Select 'Addons' tab
  • Press 'Modsets'
  • Select 'UKSF'
  • Press 'OK'
  • Check 'Addons Groups' panel contains a folder named 'UKSF'
  • Ensure the folder is checked

    •

    ACRE Setup

  • Run Arma 3 with TeamSpeak closed. ACRE will be installed automatically.

    •

    Run TeamsSpeak As Admin

  • TeamSpeak must always be run as administrator
  • As with Arma3Sync, set your 'ts3client_win64.exe' or your shortcut to run as administrator

    •

    Troubleshooting

    If you can't fix your issue, ask an SR1 for help

    Can't join server

  • Check for addons by connecting to the repository in Arma3Sync
  • Ensure there are no mods with a yellow triangle or in red text
  • Ensure you are not subscribed to anymods on the Steam Workshop that we also use
  • This is most commonly CBA_A3, ACE3, CUP_Terrains

    • Can't hear anyone/No-one can hear me

  • You need to be in the 'ACRE' channel in TeamSpeak
  • TeamSpeak MUST be run as administrator
  • Make sure the ACRE2 plugin is enabled

    •
